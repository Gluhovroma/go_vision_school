Для запуска необходимо выполнить следующие действия:
1) установить компилятор go https://golang.org/doc/install.
2) установить PATH для go, если это не произошло автоматически.
3) запуск программы осуществляется командой go run main.go.

В качестве альтернативы можно запустить исполняемый файл go_vision.exe, 
который создаст все рабочие директории и выполнит необходимые действия над файлами.

репозиторий на github: https://github.com/Gluhovroma/go_vision_school
